Latino Cultural Center Finder

By Laura Evans and Miguel Melendez

Code: lksevans12/latinoccfinder

December 2015

Since this app was our introduction to AngularJS, we wanted to keep it simple. We explored the NYC Open Data, and decided that we wanted to use an API that incorporated both of our cultures, so we picked the Latino Cultural Center API. Users can search through the different locations using keywords or location. If you press on 'show more', the information for that cultural center will appear, and the map will jump to the centers location.

The hardest part of building this app was integrating the Google Maps API. Miguel and I spent most of our time trying to figure out how to make the map jump on toggle, but after playing around with it, we were able to figure it out.

Technologies: AngularJS, Google Maps API, NYC Open Data API, CSS3

Please note that this project is not live. However, please feel free to download the code to your local machine using the link above, run it on 'python -m SimpleHTTPServer' and connect to localhost:8000.
