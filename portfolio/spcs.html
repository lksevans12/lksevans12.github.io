spcs

By Laura Evans, Sam Purcell, Zino Hu and Michael Landon

Code: lksevans12/spcs

Presentation: Watch us demo our app

December 2015

Meet spcs, the airbnb for driveways. Renters can visit spcs to rent out a provider's parking space, driveway or garage space for a fee. We used the Stripe API to allow easy transactions between users. You must sign up for Stripe before booking and creating a listing. To create a listing, simply provide details about the parking space, thhe daily, weekly and montly fees, and any other details th the renter needs to know. Providers also have the ability to specify availability using our custom built Zin Calender. Renters can search for listing by location and availability dates. Once a user has booked a listing, providers and renters can communicate through our dynamic instant messaging system built with Firebase. Users also have the ability to review and favorite, listings and other users.

Each team member had their own part of the app to build. I was responsible for the user login and authentication, implementing the ability to favorite and add ratings to users and listings, building an instant messaging system between users, and the creation and design of the profile page.

The hardest part of this project for me was storing and pulling the data from Firebase. Firebase stores the messages in a JSON object, and since I had never worked with Firebase before, some of the JavaScript was difficult to write.

In order to create or book a listing and visit a profile page, you must sign in. Sign up for your own account or feel free to use our guest login: email: guest@spcs.com password: 12345678

Technologies: Rails, PostGres, Active Record, CSS3, HTML5, JavaScript, jQuery, Bootstrap, Paperclip, Devise, Stripe, ZinCalender, Firebase, Google Maps API, AWS
