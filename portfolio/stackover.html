Stack Overabundance

By Laura Evans, Nathan Lou, Ian Lockwood and Salim Rahimi

Code: lksevans12/stackoverabundance

December 2015

This was our first project with Rails. Our goal was to make a replica of Stack Overflow.

For this project, I implemented the ability to add new questions, user login, authentication and profile page. I assited in the design of the application, and wrote tests for models and controllers. The hardest part for me was writing tests for the controllers. We used Capybara to test the features, which was a new testing framework for many of us.

Technologies: Rails, JavaScript, AJAX, Boostrap, Active Record, PostGres, RSpec, Capybara

Please note that this project is not live. However, please feel free to download the code to your local machine using the link above, run
'rails s' and connect to localhost:3000.
