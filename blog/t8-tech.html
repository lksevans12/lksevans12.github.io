U3.W8: Technical BLog: Test Driven Development

Dev Bootcamp Blog

August 7, 2015



Test driven developlment is a technique where you write a test before you write enough code. The goal of using test driven development, is to think through the requirements before writing a fully functional code, and to write clean code that works.

First you add a test, then run the tests, make changes based on the test results, run the tests again, and continue this until all tests pass.
