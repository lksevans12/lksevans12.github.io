class Height #New class of object

  def initialize (name, height, date) #instance variables
    @name = name
    @height = height
    @date = date
  end

  def intro 
    puts "Hello #{@name}. Let's measure you!"
  end

  def measure
    puts "You are #{@height}!"
  end

  def wall
    puts "------------ #{@name}, #{@height}, #{@date} has been put on the wall"
  end
end

On line 1, the code starts with a class called Height. A class is a Ruby object -- however, everything in Ruby is an objects. Objects can send and recieve messages, or methods. However, the objects need to first know what the methods are in order to understand them. We teach our methods by initializing them in classes.

On line 2, we initialized class using a method 'def initialize'. In def initialize, you will see the variables have a '@' infront of them. This is called an INSTANCE VARIABLE. An instance variable is defined in a class. It allows you to call it within different methods since it is a seperate copy of the object.
