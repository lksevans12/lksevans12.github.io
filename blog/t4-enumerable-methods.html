An enumerable method is a special methods used with the language Ruby. These methods are great for programmers because instead of having to write out each methods everytime we need it, we can just use one of the special methods.

The enumerable method known as group_by is used to group a collection by result of the block. It returns a hash where they keys are the result from the block and the values are arrays of the elements in the collection that corresponded to the key.
